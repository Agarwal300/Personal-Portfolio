# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript**. This project showcases my skills, projects, and contact information in a clean and visually appealing design.

---

## 🚀 Features

- 🎨 Modern and attractive UI design  
- 📱 Fully responsive (mobile-friendly)  
- 🧑 About Me section  
- 💡 Skills showcase  
- 🖼️ Projects section with images  
- 🔗 Social media links  
- 📄 Resume download button  
- ✨ Typing animation effect  
- 🎯 Smooth scrolling  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox & Grid)  
- JavaScript

2. Open the folder  
3. Run `index.html` in your browser
